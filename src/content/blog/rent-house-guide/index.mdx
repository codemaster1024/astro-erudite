## 看房前

1. 租房位置选择

- 离公司近
- 离公司稍远，可骑行上班
- 离公司远，需地铁 （离地铁口近）
- 离公司远，需地铁（离地铁口远）

2. 租房中介选择

- 无中介费
- 有中介费
  不管哪种，用这个中介带你看的房子跟你之前看过的进行对比（价格 环境 房子质量），综合考虑服务后决定是否在他这签约。

3. 房东选择

- 房东不好讲话的、办什么都费劲的不考虑（后期修东西 补东西估计也费劲），谈价格的时候就能感受到
- 优先一房东

## 看房

1. 尽量白天看房子，了解房间光线
2. 房子位置

3. 房子小区及周边环境(周边环境、菜市场、超市、快递站设施、地铁站距离)

4. 房子楼层以及是否有电梯

### 房间户型

**注意的是 每个东西都是要打开用一下，而不是看着有就可以的。你也可以模拟你下你日常会怎么使用的。看看是否有什么问题。**

1. 合租：几户几人合租？ 室友工作时间？室友大概年龄段？室友几人，是否有情侣？是否有宠物？
2. 朝向：优先南卧，南方夏天潮湿（北卧冬天冷 晾衣服没时候干）
3. 阳台/飘窗：看个人喜好
4. 房间实体墙/隔断：能实体墙，不要隔断，睡不好觉。
5. 房间隔音情况，在房间内播放音乐，出去听一下。卫生间淋浴放水 洗衣机洗下抹布，房间听一下。
6. 房间最好不要挨着卫生间（卫生间冲水 洗衣很吵 以及卫生问题），挨着的话就检查下声音和味道能否接受

### 房间内

1. 桌子、椅子、镜子、衣柜、空调、窗子、床、地板、灯泡 都检查一下是否破损、污渍、家电是否齐全。
2. 门缝大小，门是否能关严实
3. 灯的亮度
4. 床以及床垫自己是否喜欢，软硬 大小款式等
5. 窗户是否透漏风，开关一下
6. 空调是否好用
7. 插座位置以及是否都有电
8. 晾衣服位置
9. 衣柜打开 是否发霉破损情况。衣柜后是否漏水发霉
10. 检查房间四周的墙是否隔断，以及是否漏水

### 厨房

1. 洗漱台：水流 下水道
2. 灶台开关火力
3. 抽油烟机是否好用，是否有破损 以及噪音
4. 橱柜内部卫生以及使用情况，是否有破损

### 卫生间

1. 最好干湿分离
2. 洗漱台
3. 洗衣机：味道、排水、声音
4. 淋浴：热水器容量、淋浴水流、热力情况
5. 厕所窗户是否通风
6. 马桶使用：冲水以及卫生情况
7. 下水道排水情况

### 客厅

1. 冰箱容量是否够用，以及冰箱异味？冰箱是否漏水
2. 宽带

## 房间租金

1. 押金付几押几、合同时间
2. 物品损坏谁来修
3. 室外费用计算：
   1. 水电燃气 费用计算：民水民电？单独计费？软件计费？通过账单室友均摊？
   2. 室外水电燃气都如何计费？

## 签合同

1. 检查好上述问题确认了？
2. 保证每个物品使用过一遍了？
3. 保证自己心里确定没有不舒服的点了？
4. 合同要写明
   1. 检查租房房东信息
   2. 室内室外水电燃气的计算方式
   3. 押金以及交房租时间和金额
   4. 合同时间
   5. 违约条款
   6. 室友住房约定
   7. 其他约定好的事项，需要写进去（不要嫌麻烦，不然后面扯皮一点证据没有）
5. 中途是否可转租，以及收费方式
6. 明确水电燃气租金外的物业费 卫生费 电梯费 门卡费
7. 拍水电表使用情况

## 教训

1. 不要听房东的一面之词，他们只是想把房子租出去，你说来的问题他们的回答就是这都不是问题。所以但凡你对这个房子有问题，建议你不要当场就签，不要一头热就签，第二天签这个房子也大概率还是在的。
2. 看房记住看过的房间，防止下次看重复
3. 不要建立不必要的关系。租客和房东。
4. 楼层低的缺点：阳光问题(房间并不能全部接收到阳光，即便在中午)、蚊虫问题(容易多)、潮湿问题、临近路边会更吵。以及一楼可能是老人，容易吵到他们。
5. 模拟你下你日常会怎么使用的。看看是否有什么问题。每个东西都是要打开用一下，而不是看着有就可以的。
